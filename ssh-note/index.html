SSH 相关内容

Oct 28, 2024
1 minute read

安装

dnf list installed openssh      # 查看是否 openssh 是否安装
which ssh                       # 输出 /usr/sbin/ssh

密钥对

ssh-keygen -t ed25519 -C "" -f ~/.ssh/github_ed25519 -N ""
# ~/.ssh 如果不存在会自动创建；-N 参数设置 passphrase 为空

配置

运行ssh-add -l尝试列出当前 ssh-agent 中已经加载的私钥指纹，如果提示 Could not open a connection to your authentication agent，则：

eval "$(ssh-agent -s)"         # 启动 SSH Agent（后台进程）

再次运行ssh-add -l，如果提示 The agent has no identities，则添加私钥：

ssh-add ~/.ssh/github_ed25519  # 把 SSH 私钥添加进 SSH agent，让系统记住它

再次运行ssh-add -l，如果返回私钥的哈希和所在路径，说明添加成功，复制公钥添加到 GitHub，测试 SSH 连接：

ssh -T git@github.com
