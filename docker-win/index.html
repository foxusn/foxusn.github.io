Docker 容器运行 Windows

Jul 26, 2025
阅读时长: 1 分钟

GitHub 项目：dockur/windows，Windows 镜像：massgrave
配置文件 docker-compose.yaml：

services:
  windows:
    image: dockurr/windows
    container_name: lts11
    environment:
      DISK_SIZE: "128G"
      RAM_SIZE: "2G"
      CPU_CORES: "4"
      USERNAME: "USERNAME"
        PASSWORD: "PSW"
        LANGUAGE: "English"
        REGION: "en-US"
        KEYBOARD: "en-US"
      devices:
        - /dev/kvm
        - /dev/net/tun
      cap_add:
        - NET_ADMIN
      ports:
        - 8006:8006
        - 3389:3389/tcp
        - 3389:3389/udp
      volumes:
        - ./lts11:/storage
        - ./shared:/data
        - ./lts11.iso:/boot.iso
      restart: always
      stop_grace_period: 2m

目录结构：

~ % tree dkr-compose

    │   └── windows.ver

4 directories, 11 files

运行容器：

sudo docker compose up -d

安装 freerdp：

sudo dnf install freerdp

连接：

xfreerdp /v:localhost /u:USERNAME /p:PSW /dynamic-resolution /scale-desktop:210 +clipboard /sound:sys:alsa
