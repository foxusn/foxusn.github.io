QEMU/KVM 虚拟机安装

Jul 26, 2025
阅读时长: 1 分钟

虚拟化

检查虚拟化支持情况：

lscpu | grep -i virtualization
egrep -c '(vmx|svm)' /proc/cpuinfo
qemu-system-x86_64 --version
virsh --version

检查 KVM 模块是否加载：

lsmod | grep kvm

安装

安装软件包组 @virtulization：

dnf group info virtualization
sudo dnf install @virtulization

如果需要可选组件：

sudo dnf group install --with-optional virtualization

服务

启用 libvertd 服务

sudo systemctl enable --now libvirtd

虚拟机

通过命令安装

查看支持的系统：

osinfo-query os | grep archlinux

创建虚拟硬盘（可选）：

sudo qemu-img create -f qcow2 /var/lib/libvirt/images/arch-xxx.qcow2 20480

安装：

sudo virt-install \
--name archlinux \
--description "Arch Linux" \
--ram 4096 \
--vcpus 2 \
--cpu host \
--virt-type kvm \
--disk path=/home/$USER/VMs/arch-xxx.qcow2,size=20,bus=virtio \
--cdrom ~/Downloads/archlinux-xxx.iso \
--network network=default,model=virtio \
--os-variant generic \
--graphics spice,listen=127.0.0.1 \
--noautoconsole

sudo virsh list

sudo virsh shutdown archlinux
